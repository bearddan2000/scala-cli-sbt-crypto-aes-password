# scala-cli-sbt-crypto-aes-password

## Description
A demo for AES encryption of a password.
AES is a 256 byte encryption.

## Tech stack
- scala
- sbt

## Docker stack
- openjdk:8-jdk-alpine

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
